*  语法问题:
**  print >> f, str
- print 是将之后的内容输出到终端显示；
- print >>是将内容输出定向到某文件；
  例如：
  fpa=open("test.txt","w")
  testStr="Hello World!"
  print >> fpa, testStr
  fpa.close()
  就是将 testStr 字符串写到 text.txt 文件中。
